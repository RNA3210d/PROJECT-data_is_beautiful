# PROJECT-data_is_beautiful
Utilizing ESP32s to collect different parameters and send them to a local Influx DB server to analyze data and recognize patterns with Grafana dashboards.
- See dashboard.json for Grafana Dashboard Template
- Sensors used: DHT22(indoor), HC-SR501 PIR, BMP280(outdoor), TCS34725 RGB + Ambient Light Sensor(outdoor)
